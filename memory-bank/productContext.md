# Product Context

## 背景

`carbohydratepro` は日常管理をまとめる個人向けWebアプリである。利用者はログイン後、家計、タスク、メモ、買い物、習慣を `/carbohydratepro/` 配下で管理する。

## 主な機能

- 家計簿: 収支記録、カテゴリ、支払い方法、グラフ表示、定期支払い。
- タスク管理: スケジュール、繰り返しタスク、一時タスクのボード表示。
- メモ: Markdown対応、お気に入り、種別管理。
- 買い物リスト: 個数管理、チェック、チェック済み削除。
- 習慣トラッカー: 日次達成状態、ヒートマップ、一覧。
- 認証: メール認証、ログイン履歴、デモモード、パスワード関連機能。
- 管理: 問い合わせ、ユーザー統計、アクセス分析。

## 利用者向け方針

- 既存URL名、フォーム名、テンプレート構造は互換性の一部として扱う。
- 個人データはユーザーごとに分離される前提で、他ユーザーのデータ参照・更新を防ぐ。
- 日本語UI、`Asia/Tokyo`、数値のカンマ表示を維持する。
- 公開ページ、SEO、robots/sitemap、認証必須ページのクロール制御に注意する。

## 変更時の注意

- バグ修正では、再現条件と期待動作をテストまたは手順に残す。
- 仕様変更では、Djangoビュー・フォーム・モデル・TypeScript・E2Eの影響を確認する。
- デモモードや管理パネルは本番セキュリティに影響しやすいため、権限条件を必ず確認する。
