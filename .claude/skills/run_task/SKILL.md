# タスク実行

task.md のリストから一番上のタスクを1つ取り出し、実装を行う。

## 手順

1. `task.md` を読み込み、最初のリスト項目（`-` または `*` で始まる行）を取得する
2. タスクが存在しない場合は「タスクがありません」と伝えて終了する
3. タスクの内容を把握し、コードベースを調査した上で実装を行う
4. 実装が完了したら、`done.md` の末尾に以下の形式で記録を追加する：

```
## <タスク内容>
- **実装内容**: <実装した内容の要約>
- **実装日時**: <YYYY-MM-DD HH:MM>
```

5. `task.md` から実装済みのタスク行を削除する

## ルール

- 一度に実装するのは1タスクのみ（リストの一番上）
- 実装前にコードベースを十分に調査すること
- CLAUDE.md のルール（テスト修正、型アノテーション等）に従うこと
- done.md への記録は実装完了後に必ず行うこと
- task.md からは完了したタスクのみ削除し、他のタスクはそのまま残すこと
