# Memory Bank

このディレクトリは WSL `archlinux` 上の `/home/carbohydratepro-main` にある `carbohydratepro` プロジェクトの継続作業に必要な情報を保持する。

## 文書

- `projectbrief.md`: 目的、範囲、基本方針
- `productContext.md`: 利用背景、期待する利用者向け挙動
- `systemPatterns.md`: アーキテクチャと実装パターン
- `techContext.md`: 技術構成、環境、コマンド
- `activeContext.md`: 現在の焦点、判断、次の作業
- `progress.md`: 完了事項、未着手事項、既知の状態

## 更新ルール

- 調査や実装で事実が判明した時点で該当文書を更新する。
- 作業の区切りでは、最低限 `activeContext.md` と `progress.md` を更新する。
- 確定情報と推測は分けて記載する。
- 既存未コミット変更やユーザー作業は無断で戻さない。
